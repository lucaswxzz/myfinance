# MyFinance

MyFinance é um micro SaaS de controle financeiro pessoal focado em simplicidade, clareza e experiência real de uso.

## Visão do projeto:
A ideia é ajudar pessoas a organizarem gastos, investimentos e dívidas em um dashboard limpo, intuitivo e funcional.

## Status:
Em desenvolvimento (MVP)

## Tecnologias (atuais):
- HTML
- CSS
- JavaScript

## Próximos passos:
- Autenticação (login e sessões)
- Backend em Node.js
- Banco de dados (Postgres / Supabase)
- Estrutura escalável para SaaS

Projeto autoral com visão de longo prazo.
